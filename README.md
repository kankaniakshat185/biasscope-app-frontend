BiasScope Dashboard is the front-facing user interface for the BiasScope Intelligence platform. Designed with a striking, industrial "Neobrutalist" aesthetic, the dashboard presents complex cross-ideological media analysis in an accessible, highly visual, and interactive format.

## 🚀 Vision & Design System

The platform abandons generic, rounded, corporate UI in favor of a bold **Neobrutalist** design language. High-contrast borders, solid shadows, monospaced typography, and structural rigidity ensure that the analytical data remains the absolute focal point.

### Key UX Principles
- **No-Nonsense Analytics:** Complex NLP data is distilled into intuitive cards with strict, grid-based layouts.
- **Graceful Error Handling:** Comprehensive fallback states ensure users receive human-readable explanations if underlying datasets are empty or API rate limits are hit.
- **Interactive Discovery:** Users can drill down from high-level events directly into the "Raw Claims" and the original publisher sentences backing those claims.

## 🛠️ Tech Stack
- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS + Shadcn UI (Customized for Brutalism)
- **Authentication:** Better Auth (Secure, session-based auth)
- **Data Fetching:** Native `fetch` with Next.js App Router caching.
- **Icons:** Lucide React
- **Hosting:** Vercel

## ✨ Core Features

*   **🛡️ Mandatory Authentication Gate**
    *   Secure, robust login barrier protecting the analytical platform using Better Auth.
*   **📊 Claim Explorer & Intelligence Layer**
    *   Displays macro-events generated by the backend clustering engine. Highlights "Core Claims", Evidence Counts, and Source distributions.
*   **✅ Cross-Ideological Consensus Badges**
    *   Visually flags claims that are backed by a high `consensusScore`—meaning they are corroborated by both left-leaning and right-leaning outlets.
*   **🗣️ Contrastive Echo Chambers (`BETA`)**
    *   Side-by-side comparative cards (Red vs Blue) showing the LLM-generated narrative framing of the Left-wing and Right-wing media ecosystems for a specific topic.
*   **📈 Entity Sentiment Graph (`BETA`)**
    *   Grid visualization showing exactly which named entities (People, Organizations) are driving the most positive or negative sentiment across the coverage.
*   **🔎 Search & History Vault**
    *   Real-time topic search integration and a dedicated "Vault" page allowing users to view, filter by category, and delete their previous analytical snapshots.

## 🚧 Ongoing Improvements
- **Infinite Scroll / Pagination:** Implementing continuous scrolling for the "Analyzed Articles" feed to support massive datasets without performance degradation.
- **Dynamic Methodology Rendering:** Transitioning the static methodology report to dynamically reflect the validation metrics returned by the backend.
- **Real-Time WebSockets:** Adding live-loading bars during the 20-30 second backend NLP processing phase.

## 💻 Local Development Setup

### 1. Prerequisites
- Node.js (v18+)
- npm or pnpm
- Access to the BiasScope Backend API

### 2. Installation
```bash
git clone https://github.com/kankaniakshat185/biasscope-app-frontend.git
cd biasscope-app-frontend
npm install
```

### 3. Environment Variables
Create a `.env.local` file in the root directory:
```env
NEXT_PUBLIC_API_URL="http://127.0.0.1:8000"
BETTER_AUTH_SECRET="your-auth-secret-key"
NEXT_PUBLIC_APP_URL="http://localhost:3000"
```

### 4. Running the Development Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
